User Stories from Product Backlog that we will address in this sprint:

a) As a user, I want to be able to find all files with a given name inside the pathway provided so that I can check if a file exists.
b) As a user, I want to be able to find all the directories with a given name inside the pathway provided so that I can check if a directory exists.
c) As a user, I want to be able to save the output in a file instead of terminal so that I can view it later.
d) As a user, I want to be able to retrieve a file from a given URL so that I can store it in my working directory.
e) As a user, I want to be able to save my current file system, so that I retrieve it later
f) As a user, I want to be able to load a previous file system so that I can continue working on it.
